## Description
This project automates Flipkart product selection.

## Features
- Open Flipkart and search product
- Select first product
- Count available offers
- Add to cart
- Take screenshot

## Prerequisites
- Java
- Selenium WebDriver
- ChromeDriver

# How to run
1. Open project in Eclipse
2. Go to: src → Flipkart → T_01.java
3. Right click → Run As → Java Application
4. Browser will open automatically
5. Output will be shown in console
6. Screenshot will be saved in Screenshot folder

## Output
- Console: Number of offers
- Screenshot: cart.png
